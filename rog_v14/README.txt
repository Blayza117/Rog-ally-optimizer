ROG Ally Optimizer v14

HOW TO RUN:
  Double-click   Launch.bat
  (or right-click ROGAllyOptimizer.ps1 → Run with PowerShell)

No Python. No installs. Runs on any Windows 10/11 PC.

FEATURES:
  Simple Mode    — Xbox-style game tile launcher. Pick a game, hit PLAY.
  Advanced Mode  — Full settings: resolution, power mode, priority, AI notes.
  AI Optimize    — AI tunes every game for your specific Ally model.
  Scan Games     — Auto-detects Steam, Xbox, Epic, GOG, EA, Ubisoft.
  Bloat Remover  — One-click remove junk slowing your Ally.
  Clean RAM      — Kill background apps before gaming.
  Xbox Mode      — Detects and works with ASUS Xbox Mode.
  Handheld UI    — Big touch targets, gamepad navigation (toggle HANDHELD).
  Export/Import  — Share your game profiles with other Ally owners.

GAMEPAD NAV (Simple Mode):
  Ⓐ = Play   Ⓑ = Back   Ⓧ = Scan   Ⓨ = Add
  ▲▼◀▶ = Navigate   LB/RB = Scroll

FIRST TIME:
  Your profiles save to profiles.json in the same folder.
  The AI optimize button requires an internet connection.
